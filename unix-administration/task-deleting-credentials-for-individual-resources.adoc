= 刪除個別資源的認證資料
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以刪除任何一項安全資源的認證資料、例如設定檔、儲存庫或主機。這可讓您只移除一項資源的認證、而非清除所有資源的使用者認證。
