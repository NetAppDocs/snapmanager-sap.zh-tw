= 刪除設定檔的使用者認證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以刪除設定檔的使用者認證、讓使用者無法再存取。

. 若要刪除使用者的設定檔認證資料、請輸入以下命令：「shmsap認證刪除-profile -nameprofile_name」
