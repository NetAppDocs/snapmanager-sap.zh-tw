= 檢視與設定檔相關之特定作業的詳細記錄
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢視SnapManager 與設定檔相關的特定故障操作詳細記錄。

.步驟
. 若要顯示SnapManager 與設定檔相關之特定功能的詳細歷史記錄資訊、請輸入下列命令：
+
「msap history operation-show -profile profile_name｛-label label label |-id id｝[-quiet |-verbose]」
