= 刪除設定檔的使用者認證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以刪除設定檔的使用者認證、讓使用者無法再存取。

.步驟
. 若要刪除使用者的設定檔認證、請輸入下列命令：
+
‘* smsap認證刪除-profile -name _profile_name_*’
