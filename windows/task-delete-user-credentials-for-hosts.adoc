= 刪除主機的使用者認證資料
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以刪除主機的認證資料、讓使用者無法再存取。此命令可讓您只移除一項資源的認證、而非清除所有資源的所有使用者認證。

.步驟
. 若要刪除使用者的主機認證、請輸入下列命令：
+
「msap認證刪除-host -name_host_name_-username_-username_」
