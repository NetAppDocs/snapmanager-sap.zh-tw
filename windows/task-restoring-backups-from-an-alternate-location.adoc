= 從替代位置還原備份
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從替代位置還原資料檔案、將資料檔案從中繼檔案系統還原至作用中檔案系統。

* 建立還原規格XML檔案、並指定您要使用的還原方法類型。


您可以使用smsap備份還原命令、並指定您建立的還原規格XML檔案、以便從替代位置還原備份。

. 輸入下列命令：
+
「msap備份還原-profileprofileprofile-labelLabel完整-alllogs還原程式」
