= 刪除備份排程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在不再需要備份排程時、刪除這些排程。

. 若要刪除備份排程、請輸入以下命令：「msap排程刪除-profile profile_name-schedule-namesschedulName（-quiet |-verbose）」
