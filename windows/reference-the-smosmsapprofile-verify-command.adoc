= smsap設定檔驗證命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以執行「profile VERIFY」命令來驗證設定檔的設定。您必須先掛載資料庫、才能執行此命令。



== 語法

[listing]
----

smsap profile verify
-profile profile_name
[-quiet | -verbose]
----


== 參數

* *
+
指定要驗證的設定檔。設定檔包含資料庫和其他資料庫資訊的識別碼。

* *（靜音）*
+
僅在主控台顯示錯誤訊息。預設為顯示錯誤和警告訊息。

* *（冗長）*
+
在主控台顯示錯誤、警告和資訊訊息。
